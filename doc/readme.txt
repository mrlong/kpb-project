1、	客户端是自动应答服务器查询的装置。查询命令格式，和数据格式都是固定的。
    与在局域网内有两台PC进行TCP通讯没有区别。因此，你也可以通过在局域网内有两台PC测试。
2、	查询命令用十六进制发命令为： 000224303052503033320D
回复数据为: *16+00.02535  ( 前后都不要，取值0.025，SQL中保留2位小数为float,8字节,)

3、	客户端每两分钟发一个$的字符，用于维持通信。无需记录。

4、	客户端一旦建立连接，会发一个标识符，如xs,#1001，这样，所有与该客户端的数据记录到xs,#1001设备。

5、	客户端如果断电。会自动重新建立一个连接，由于不是固定IP，会生成新的IP地址和端口号。

6、建议主表格式为：STCD（客户端ID 取1001 1002 ）
